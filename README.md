# Food Fest Ticket Landing Page

## Project Overview

This project is a Vue 3 single-page application developed for Food Fest 2026.

The landing page showcases different ticket tiers using reusable Vue components. Users can view ticket information, identify featured packages, and interact with a favourite feature that simulates engagement found in modern e-commerce websites.

---

## Features

- Dynamic rendering of ticket tiers
- Reusable TicketCard component
- Featured ticket highlighting
- Favourite interaction
- Responsive layout for desktop and mobile
- Vue 3 Composition API
- Computed properties
- Props-based component communication

---

## Technologies Used

- Vue 3
- Vite
- HTML
- CSS
- JavaScript

## Installation
- Clone the repository:
    - https://github.com/AngelaSolomons/food-fest-ticket-landing-page.git
    
- Navigate into the project:
    - cd food-fest-webpage

- Install dependencies:
    - npm install

- Run development server:
    - npm run dev
 
- View project:
    - copy localhost link
    - paste link in browser
